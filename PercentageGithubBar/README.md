# PBG - Percentage Github Bar
## v1.0

What it is:

A tool that calculates the percentage of a language in a repo with the number of files in the repo.

Why it matters:
The default Github bar in every repo calculates the percentages of a language with the size of the files in bytes, and I wanted to do it with the number of files.

What it does differently:

**It doesn't require any Github Personal Access Token.**
