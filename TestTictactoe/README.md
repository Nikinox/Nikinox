**Blindly vibe coded Tic tac toe game.**

### Tools I used:
- Ollama
- Qwen 2.5 coder 7b

### The prompt I used:

        build a tic tac toe in HTML with CSS and JS built inside the HTML file

### How to install it:

- download the .html file
- run it inside your browser

### What I learned:

Local LLMs (Especially the small ones) are much more comfortable coding in frontend, so JS, HTML and CSS.

### Notes:

Of course it could be improved, I just wanted to test local LLMs capabilities in coding simply.
