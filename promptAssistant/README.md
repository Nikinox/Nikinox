🧠 Prompt Builder CLI

A simple yet powerful interactive prompt generator for the terminal, designed to help users build structured, high‑quality prompts for conversational AIs.
The program guides the user step by step, collects all required information, and outputs a complete prompt ready to paste anywhere.
🚀 Key Features

  Interactive loop: the program keeps generating prompts until the user decides to stop.

  Guided input: each part of the prompt is requested through clear, targeted questions.

  Input validation: yes/no questions accept only valid responses.

  Clean output: at the end of each cycle, the program prints the full prompt in a logical order.

  Flow control: the user chooses whether to copy the prompt and whether to continue generating new ones.

📂 Prompt Structure

The final prompt is composed of six essential sections:

    Role – the role the AI must assume

    Context – background information and scenario

    Task – the specific task the AI must perform

    Format – the required output format

    Rules – boundaries and elements to avoid

    Examples – examples provided by the user (text or attached files)

This structure ensures clarity, precision, and consistency across all generated prompts.

🛠️ How to Use

Run the script from your terminal:
Code

    python [name of the file].py

  Answer each question as prompted.

  When asked, choose whether to copy the generated prompt.

  Decide whether to continue generating new prompts.

  Paste the final prompt into any AI system you want to use.
