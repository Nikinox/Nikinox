🕹️ Pygame Movement Template

A minimal, clean starter template for building a 2D game using Pygame.
This project includes:

    a resizable game window

    event handling

    smooth player movement

    support for both WASD and arrow keys

    a simple, readable game loop

You can use this file as a base to copy when starting new Pygame projects.
🚀 Features

    Resizable window (pygame.RESIZABLE)

    Keyboard input handling (WASD + Arrow Keys)

    Player movement using Rect.move_ip()

    Screen clearing and redrawing each frame

    Minimal structure ready to be expanded into a full game

📦 Requirements

Make sure you have Python 3.x and Pygame installed:
bash

    pip install pygame

🧱 Using This as a Game Template

This file gives you a ready-to-use structure for:

    adding sprites and animations

    implementing collisions

    creating tile maps

    adding enemies or NPCs

    building menus and HUDs

    adding physics or gravity

The main loop (while running:) is already set up for expansion.
