🎮 Sheep Paper Tycoon

A small tycoon game where you manage a sheep farm and you sell paper.

📝 Description

This is a small game developed with TurboWarp.

The player manages a farm, produces resources, and tries to keep everything balanced.

The concept originated from a spontaneous idea that emerged during a history class with the fact that in Pergamo they used to make paper with sheepskin.

🧩 Main Features

    Dynamic customer system with random arrivals and departures

    Procedural pedestrians with color variations

    Resource production and management

🚀 How to Play

Open one of the links between https://scratch.mit.edu/projects/1321202030/ and 

The game will automatically start in your browser

🛠️ Technologies Used

    TurboWarp Editor

    TurboWarp Packager

    Scratch 3.0 (block‑based language)
